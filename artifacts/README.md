# Monero_c Build Artifacts

This folder contains pre-compiled binary artifacts for the `monero_c` library used by Skylight Wallet.

### Source & Provenance
These binaries are automatically built from source using GitHub Actions to ensure a transparent and verifiable supply chain.
- **Source Workflow:** [.github/workflows/build-monero-c.yml](../.github/workflows/build-monero-c.yml)
- **Build History:** [GitHub Actions Runner History](https://github.com/MAGICGrants/skylight-wallet/actions/workflows/build-monero-c.yml)

### Reproducibility
You can reproduce these builds locally by running the commands defined in the workflow file linked above. This ensures that the binaries match the open-source code in this repository.
